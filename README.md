# 📊 Data Science & Analytics Internship Tasks

## 🏢 DevelopersHub Corporation Internship

This repository contains solutions to 3 core data science tasks completed as part of the internship program. The tasks demonstrate skills in data analysis, visualization, and machine learning using Python.

---

# 📌 Tasks Overview

## ✅ Task 1: Iris Dataset Exploration & Visualization

* Loaded dataset using **pandas**
* Explored dataset structure (`shape`, `columns`, `head`)
* Created visualizations:

  * Scatter plots
  * Histograms
  * Box plots
* Identified patterns and outliers

📊 **Key Insight:**
Petal features clearly differentiate flower species.

---

## ✅ Task 2: Credit Risk Prediction

* Cleaned dataset (handled missing values)
* Performed Exploratory Data Analysis (EDA)
* Applied **Logistic Regression**
* Evaluated model using:

  * Accuracy
  * Confusion Matrix


---

## ✅ Task 3: Customer Churn Prediction

* Preprocessed dataset (encoding categorical variables)
* Built model using **Random Forest Classifier**
* Evaluated performance using:

  * Accuracy
  * Confusion Matrix
  * Classification Report
* Analyzed **feature importance**

📊 **Key Insight:**
Customer churn is highly influenced by **age, balance, and geography**.
